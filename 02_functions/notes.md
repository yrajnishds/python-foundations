# 02 - Functions in Python

## 1. What is a Function?

A function is a reusable block of code that performs a specific task.

Functions improve:
- Modularity
- Code reusability
- Readability
- Maintainability

---

## 2. Syntax

def function_name(parameters):
    """Optional docstring"""
    # code
    return value

---

## 3. Types of Functions

1. Built-in functions (print, len, sum)
2. User-defined functions
3. Lambda functions

---

## 4. Parameters & Return Values

Parameters → Input to function  
Return → Output from function

---

## 5. Variable Scope

- Local variables
- Global variables

---

## 6. Best Practices

- Use meaningful names
- Keep functions small
- Write docstrings
- Avoid global variables
