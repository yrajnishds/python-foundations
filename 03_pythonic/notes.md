# 03 - Pythonic Programming & File Handling

## 1. What is Pythonic Code?

Pythonic code means:
- Simple
- Readable
- Concise
- Idiomatic

Examples:
- List comprehensions
- Using built-in functions
- Proper exception handling

---

## 2. File Handling Modes

'r'  → Read  
'w'  → Write (overwrite)  
'a'  → Append  
'rb' → Read binary  

---

## 3. Exception Handling

try:
    code
except:
    handle error

Common Exceptions:
- ValueError
- FileNotFoundError
- ZeroDivisionError

---

## 4. JSON Handling

Used for:
- Data exchange
- Configuration files
- APIs

Use:
import json
json.load()
json.dump()
